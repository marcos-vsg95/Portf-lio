    Agosto - 2025

    Projeto Redes Sociais

    Esse projeto teve como objetivo criar uma página simulando a tela de um smartphone, com um visual semelhante ao iPhone. A estrutura foi feita utilizando HTML e CSS, com um iframe inserido dentro da moldura do celular para carregar o conteúdo de cada “aplicativo”.

    Saiba mais...

    Na lateral, adicionei botões com ícones das principais redes sociais (Instagram, GitHub, LinkedIn, Spotify e YouTube). Ao clicar em cada ícone, o conteúdo exibido na tela do celular muda, criando uma interação mais dinâmica com o usuário.

    Trabalhei também com efeitos de hover, sombras e responsividade para que os botões se destaquem visualmente. O fundo de madeira traz um contraste interessante que ajuda a destacar o dispositivo no centro da página.

    Setembro - 2025

    Projeto Tela de Login Responsiva

    Neste projeto, desenvolvi uma interface de login moderna e totalmente responsiva, utilizando apenas HTML5 e CSS3. A proposta foi criar um layout elegante, com barras arredondadas, ícones nos campos de entrada e uma imagem de destaque na parte superior, tornando a página visualmente mais interessante.

    Saiba mais...

    O formulário conta com validações básicas de e-mail e senha, além de opções como botão de acesso e link para recuperação de senha. A estrutura se adapta automaticamente a diferentes tamanhos de tela, graças às media queries, alterando o posicionamento dos elementos conforme o dispositivo utilizado (celular, tablet ou desktop).
