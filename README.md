# 💼 Portfólio - Marcos Vinícius

Este é o meu portfólio pessoal desenvolvido com foco em apresentar meus projetos, habilidades e evolução como desenvolvedor.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* API do GitHub

## ✨ Funcionalidades

* 🌙 Alternância entre tema claro e escuro
* 📱 Layout responsivo
* 🔄 Listagem dinâmica de projetos utilizando a API do GitHub

* 🔗 Links diretos para repositórios e deploys

* Boas práticas de organização de código
* Estruturação de projetos front-end
* Uso de Git e GitHub no fluxo de desenvolvimento
